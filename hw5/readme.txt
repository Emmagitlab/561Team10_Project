Keynote:

1. To create the collection, use the file createDB.js so it will drop old "test" collection and create new one
   > mongo
   > load("createDB.js");

2. to execute the CRUD, use the file hw5.js
   > load("hw5.js");

3. to check the output result, use the following command line step by step in your terminal
   > script hw5output.txt
   > mongo
   > load("createDB.js");
   > load("hw5.js");
   > quit();
   > exit;

   After finish, all output results will be in hw5output.txt
